# qasemrush

### Tests launch ###

You'll need java 8 or higher, Maven 3.6.3

JAVA: https://www.oracle.com/ru/java/technologies/javase-downloads.html

MAVEN: http://maven.apache.org/

1. Open cmd.

2. In cmd go to folder with tests:

    write proper path to folder with qasemrush project for example:
```
cd C:\folder\qasemrush
```
    after that execute this command.

3. Execute a command:
```
mvn clean install
```

#### OR ####


You can run tests in Intellij IDEA

1. Open folder qasemrush as idea project

2. To run tests use: 
```
 mvn clean install 
```
